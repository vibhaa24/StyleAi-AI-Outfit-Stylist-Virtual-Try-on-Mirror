# 👗 AI Outfit Stylist + Virtual Try-On Mirror

## 📌 Project Overview

This project is an AI-based outfit recommendation system that suggests clothing based on user preferences and allows virtual try-on using AR technology.

---

## ✨ Features

* AI-based outfit recommendation
* CNN model for clothing classification
* Virtual try-on experience
* Dataset-based fashion catalog
* Interactive UI using Flask

---

## 🛠 Tech Stack

* Python (Flask)
* TensorFlow / CNN
* HTML, CSS, JavaScript

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python app.py
```

Then open:
http://127.0.0.1:5000

---

## 📂 Project Structure

* `app.py` → main backend
* `templates/` → frontend HTML
* `static/` → CSS & JS
* `train_cnn.py` → model training
* `prepare_dataset.py` → dataset processing
